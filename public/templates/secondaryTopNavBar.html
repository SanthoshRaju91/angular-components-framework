Secondary Top Navigation Bar

This components provides a secondary navigation bar right below the main / primary navigation.
This is an alternate for side navigation bar menu.

Usage

HTML directive
  • menus - list of secondary menu items (example provided below)
  • top - Boolean specifying whether the secondary nav bar should stick below the top nav-bar or not.
Scope Object to be placed in the controller - list of secondary-top-nav-bar menu items.
$scope.secondaryMenuList = [{ name: "Weather Widgets", iconClass: "", url: "" }, { name: "Chart Widgets", iconClass: "", url: "" }, { name: "Statistics", iconClass: "", url: "" }, { name: "Maps Integration", iconClass: "", url: "" }, { name: "Work Widgets", iconClass: "", url: "" }];
