Top Navigation Bar

Top Navigation Bar components allows you to create a top bar navigation for your page, with stated based (angular-ui-router) transistions.
Also you can integrate setting menu, customization pages and logout options for user and user profiles too.

Usage

HTML directive
  • options - list of menu configuration (example provided below)
  • top - Boolean specifying whether the nav bar should stick to the top or not.
Scope object - configuration data for top-nav-bar (Place this in your main angular controller)
$scope.navData = { logo: { name: "Markers", image: "http://getbootstrapadmin.com/remark/material/topicon/assets/images/logo-blue.png" }, search: true, notification: { show: true, iconClass: "fa fa-bell-o" }, messages: { show: true, iconClass: "fa fa-paper-plane-o" }, user: { show: true, image: "http://getbootstrapadmin.com/remark/material/global/portraits/5.jpg", menuOptions: [{ iconClass: "fa fa-cogs", name: "Settings", toRoute: "" }, { iconClass: "fa fa-sliders", name: "My-Page", toRoute: "" }, { iconClass: "fa fa-sign-out", name: "Logout", toRoute: "" }] } };
