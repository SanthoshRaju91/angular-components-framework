Project Scaffolding

There is scaffolding structure which can be downloaded from the below button.
This has a complete structure of the application, all the *.js, *.css, *.html code should go to the /public folder.
Also, comes with a complete task automation, both for development and production.
In case of development, the gulp file has a watch on all the *.scss, *.js & *.html files, so when the file is changed, the development build process starts automatically by compiling the scss files to css files and js files to check for any synatic errors. And more, you don't have to reload the browser to view your changes, the gulp files automatically detects the changes made to any of the files and reloads the browser.

Where as in case of production, there is small build process, where the *.css and *.js files are concatenated and minified and creates a /build folder containing the final production build which can be deployed in project server serving static files.

Instructions to setup

  • Download the Project Scaffolding by clicking the below button.

    Download Project Folder

    Or Clone it via git git clone https://github.com/SanthoshRaju91/projectScaffolding.git
  • Then install the bower and node modules required to start the development. (Please have node.js, gulp & bower installed)

    npm install; bower install;
  • Run the below gulp command to start the development

    gulp (or) gulp default
  • Run to the below gulp command for production below

    gulp build
That's all folks, you now have a both production and development environment setup in about 10-ish minutes.
