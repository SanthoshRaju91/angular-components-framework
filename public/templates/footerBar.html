Footer Bar

This component provides a bottom fixed footer bar to be loaded to your page.
Component can also be configured with social media links, which will be provided in the configuration data.

Usage

HTML directive
  • data - configuration object to be provided to the footer directive, we can also have social media configuration. (example is given below)
  • bottom - Boolean specifies whether the footer should be stuck to the bottom of the page.
Scope object from the controller - which is the configuration object to the footer directive.
$scope.footerData = { brand: "Makers", author: "Team Oct", socialMedia:[{ name: 'facebook', iconClass: 'fa fa-facebook', route: '' }, { name: 'twitter', iconClass: 'fa fa-twitter', route: '' }, { name: 'google', iconClass: 'fa fa-google-plus', route: '' }] };
