Angular Common Components.

This framework is built for easily developing applications with an ease.
Built with some of the commonly used components for building application, we have solidated them as a custom directives with a Project Scaffolding with DEV and PRODUCTION build process.

This project was built keeping in mind of building production application very easy.
This comes with Charts directive and maps integaration, so building a Dasboard / Analytics Dashboard is as easy as building a lego.

Please check the list of components on built Left Navigation with documentation to integrate the custom directives.

Components used

  • Angular 1.5
  • Angular UI Router
  • Bootstrap
  • Gulp - for build and task automation

The list will be updated with additional components very soon. In case of any issues, please raise a bug.
