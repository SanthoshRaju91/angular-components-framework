Charts Widget Component

Could be used for building Charts very easily, with this directive creating a dashboard with awesome charts is just a childs play.
Please note, more charts and feature will be added to this component in the future. For now we have integrated all types of basic charts need to build a complex dashboard. More complex charts and its configuration are to come, so please be patient.

Usage

HTML directive
  • data - configuration data with the type of chart to be rendered to be given. Please note this should be given in the controller.(example below)
Scope object, having the configuration data to render the chart.
$scope.chartData = { columns: [ ['data1', 30, 200, 100, 400, 150, 250], ['data2', 50, 20, 10, 40, 15, 25] ], type: 'line' };
